Local Storage Example using Associative Array

Also CSS Transform, Transition and Animation Examples



This example takes your input and:
1. Creates an associative array.
2. Clicking "Save" puts the array into local storage using setItem, and then calls it back to display in JSON format using getItem
3. Clicking "Remove First Name" deletes the first name from local storage using removeItem, then displays the remaining values
4. Clicking "Clear All" removes all items from local storage using the .clear command, and then shows that those values are no longer stored.
