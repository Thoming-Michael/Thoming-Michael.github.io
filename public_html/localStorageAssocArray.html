Local Storage Example

This example takes your input and:
1. Clicking "Save" puts it into local storage using setItem, and then calls it back to display using getItem
2. Clicking "Remove First Name" deletes the first name from local storage using removeItem, then displays the remaining values
3. Clicking "Clear All" removes all items from local storage using the .clear command, and then shows that those values are no longer found.
